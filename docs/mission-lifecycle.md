# Mission Lifecycle

## Stages

1. Inspect environment
2. Inspect workspace
3. Plan
4. Confirm approvals
5. Execute skill chain
6. Validate evidence
7. Persist runtime artifacts
8. Report

## Approval Gates

Approval is required for:

- wallet generation
- keypair changes
- cluster changes
- airdrops
- deploy actions
- mainnet actions
- upgrade actions
- destructive file operations
