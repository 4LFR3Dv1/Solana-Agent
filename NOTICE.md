## Architectural provenance

The command-authority, governed-lifecycle, ledger, mission-pack, and
evidence-first concepts used by this project were informed by the SNE Foundry,
which was developed by the same project team.

Solana Agent does not copy the SNE Foundry product as a whole. Those concepts
are being independently scoped and implemented for a small, local-first,
Solana-native runtime. The Foundry backend, cockpit, authentication, commerce,
remote-access, and institutional-memory layers are not part of this project.

Future code or assets derived from third-party projects must be recorded in
this file together with their applicable licenses.
