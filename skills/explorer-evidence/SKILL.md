# Skill: explorer-evidence

## Purpose

Assemble a mission evidence pack suitable for demos, bounties, and grant proof of work.

## Preconditions

- deployment or invocation data exists

## Inputs

- deployment record
- invocation artifacts
- mission summary

## Procedure

1. collect Program ID, cluster, wallet address, and relevant signatures
2. construct Explorer links
3. write a human-readable evidence pack
4. persist an artifact index

## Required Outputs

- evidence pack path
- normalized artifact list
